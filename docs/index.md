# How to reset my password?

/// admonition | Don't call the help desk
    type: warning

Do not open a ticket at MyIT help desk if you have forgotten you password. They can't help you and will charge Siemens a lot of money.
///

## Self-service

When you read this text you have most likely forgotten your Microsoft Account password.

![Forgot my password](./Enter%20password.png "What's my password?")

The best way to proceed—which works in 99.9% of all cases—is to click on `Sign in another way`.

![Sign in another way](./Enter%20password%20-%20sign%20in%20another%20way.png "Alternative sign in")

This will bring you to this page where you can select passwordless methods authenticate. If you have registered MS Authenticator on your phone or if you own a PKI card, you can use those devices to sign in.

![Select PKI or Authenticator](./Sign%20in%20with%20PKI%20or%20Authenticator.png "Go passwordless")

If this works, please [request a new password in MyIT](https://myit.siemens.com/myitportal?id=sc_cat_item&table=sc_cat_item&sys_id=29738742c3025ad423c3dc4c0501311c). The necessary steps are documented in a [manual](https://manuals.siemens.com/useit/manual/useit/en/how-to-reset-your-user-account-password-via-myit).

## Assisted process

### Manager driven

In the rare case the self-service isn't working, please reach out to your manager directly (or super-requester if your unit has implemented this concept). The manager (or super-requester) must open MyIT and select the menu item `Act on Behalf of`.

![Act on behalf](./Act%20on%20behalf%201.png "Select `Act on Behalf of`")

In the following dialog your manager (or super-requester) must select you by clicking on your name.

![Select employee](./Act%20on%20behalf%202.png "Click on name")

If successful the manager (or super-requester) should see the MyIT landing page with a banner informing about a successful impersonation.

![Success](./Act%20on%20behalf%203.png "Banner shown").

Now the manager can follow the above-mentioned processes to reset a password.

### Emergency reset

If the manager can't be reached there is an emergency process that can be performed by typically two colleagues together. How to run this process is documented in a [manual](https://manuals.siemens.com/useit/manual/agm/en/emergency-password-reset).
